移除Chrome表单自动填充时默认样式的几种方案

发表于 2018-05-05 | | 阅读次数:

Google Chrome浏览器在进行表单自动填充时，会自动给输入框添加上默认的样式，这黄底黑字如果不在意也就那么回事儿，但如果真纠结起来，怎么看怎么不顺眼，稍微查了一下移除默认样式的方案，发现还挺涨姿势的。

表单在自动填充后，真正起效果的其实是:-webkit-autofill这个伪类，对应的样式是这样的：

最直接的想法自然是用自定义的样式将其覆盖掉，然而实际上并不可能这么简单，即使使用!important去提权值，也无法覆盖掉这几个属性，因而，解决思路主要是从视觉效果上，使用其他样式将该默认样式覆盖掉。

方案一

使用-webkit-box-shadow代替background-color，通过修改第四个属性spread适应不同输入框的大小，也可以修改第五个属性color自定义颜色，不过需要注意不能使用transparent
